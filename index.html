HTML

  • HTML: Hyper Text Markup Language
  • The head element contains information about the webpage, known as meta data and contains meta tags.
  • The body element represents the visible content shown to the user.
  • The li tag is used for individual items and they are encapsulated inside ul or ol tags

CSS

  • CSS: Cascading Syle Sheets
  • Three types: inline, internal style sheet, and external style sheel
  • div { color: blue; } div is the selector, color is the prooperty, and blue is the value
  • the selector determines which assets the css will apply to. you can use a single element, comma seperated list, or a wildcard
  • you can use a class attribute as a selector. the class attribute is noted by a preceding dot. any elements with this class attribute will have the css code applied to it
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: be sure to do a pull of main branch before working in a new branch
  • git add -A: add changes from the branch to the staging area
  • git commit -m "Notes": push the changes from the staging area to the repo and add a note

JavaScript

  • Code that allows users to interact with a website
  • console.log is used to print messages to the console
  • the code is read line by line, from top to bottom so th order matter. known as "order control"
  • for loops are used to repeat code blocks. the code is repeated until the condition in the for loop is false
  • exampe of for loop: for (var x=0; x < arrayname.length; x++) { /* code block; */ }
